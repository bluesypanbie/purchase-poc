# Figma 디자인 레퍼런스

이 폴더는 Figma에서 직접 복사한 코드 파일을 보관하는 곳입니다.

---

## 용도

- Figma에서 디자인한 UI의 코드를 그대로 복사해서 여기에 저장
- 이 코드의 스타일, 구조를 참고하여 Framer용 코드 컴포넌트로 변환

---

## 파일 형식

- `.tsx`, `.jsx` - Figma에서 내보낸 React 코드
- `.svg` - 아이콘, 그래픽 에셋
- 비트맵 이미지 (`.png`, `.jpg` 등)

---

## 작업 흐름

1. Figma에서 UI 디자인
2. Figma Dev Mode 또는 플러그인으로 코드 복사
3. 이 폴더에 파일로 저장
4. 해당 코드를 참고하여 `reference/framer/` 폴더에 Framer용 컴포넌트 작성
